Debugging

You can debug using this commands in Browser Console

require('uiRegistry').get(function(component){console.log(component.name);})

This command will list ALL available KO JS components in the current page.

For example, in this list you will find component_scope which is the name of this component

require('ko').contextFor($0)

If you DOESN'T know the name of the Component in the window, you can click right click -> Inspect, then type this command in console and the information about the clicked element will be displayed

require('uiRegistry').get('component_scope')

This command will show you all the information about 'component_scope' Component

require('uiRegistry').get('component_scope').numberOfClicks()

This command will show you the value of the observable "numberOfClicks"
